#  E-commerce Amigurumi - Frontend

Interface web em Angular para loja de amigurumi com finalização de pedidos via WhatsApp.

##  Sobre o Projeto

Sistema de e-commerce onde o cliente navega pelo catálogo, adiciona produtos ao carrinho e finaliza o pedido diretamente pelo WhatsApp.

##  Tecnologias

- Angular 17 (Standalone Components)
- TypeScript
- JWT (autenticação do admin)
- HTML + CSS

##  Como rodar

1. Clone o repositório:
```bash
git clone https://github.com/vinigit22/amigurumi-frontend.git
```

2. Instale as dependências:
```bash
npm install
```

3. Certifique-se que o backend está rodando em `http://localhost:3000`

4. Rode o projeto:
```bash
ng serve
```

5. Acesse no navegador:
```
http://localhost:4200
```

##  Funcionalidades

-  Catálogo de produtos
-  Carrinho de compras
-  Finalização de pedido via WhatsApp
-  Área admin com login JWT
-  Cadastro e edição de produtos
-  Relatório mensal de produtos

##  Acesso Admin
```
Rota: /login
```
Credenciais configuradas diretamente no banco de dados.
